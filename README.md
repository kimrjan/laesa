# LAESA index implementation
Semestral project in subject BI-VWM (Searching Web and Multimedia Databases).
Project destription is in `doc/zaverecna_zprava.pdf`.

## Run

Prepare environment
```
conda env update
conda activate laesa
```

notebook
```
python3 -m notebook
```

web GUI
```
python3 -m http.server
```
